# Container Loading

## Procedure

1. Verify booking confirmation.
2. Inspect container condition.
3. Verify container seal.
4. Confirm weight declaration.
5. Load container according to stowage plan.
6. Confirm loading completion.

## Safety

Never load damaged containers without inspection approval.
