# Reefer Operations

## Purpose

This SOP explains the handling of refrigerated containers.

## Procedure

1. Verify temperature settings.
2. Connect reefer container to power.
3. Monitor temperature every four hours.
4. Record temperature deviations.
5. Escalate deviations greater than 2°C.

## Escalation

Notify Reefer Operations immediately if temperature remains outside the acceptable range for more than one hour.
